# Getting Started with Buntal and MDX

This is a comprehensive guide on how to build a blog with **Buntal** framework and **MDX** for content management.

## What is Buntal?

Buntal is a modern web framework built on top of Bun runtime, providing:

- Fast development experience
- Type-safe routing
- Built-in server-side rendering
- Excellent TypeScript support

## MDX Integration

With Bun's macro system, we can load MDX content at compile time for better performance:

```typescript
import { mdx } from './render.tsx' with { type: 'macro' }

const Post = mdx('./example-post.mdx')
```
